# uBraids
Another version of Mutable Instruments Braids, inspired by MagpieModular's work

Visit the Very Good Indeed project page over here: https://github.com/MagpieModular/microBraids to see the inspiration for this module.
I've reduced the complexity of the schematic, dumped a couple of the chips, and I can't remember what else right now.

The code to turn the 14-segment display data into something on an OLED screen is what I'm working on right now.
